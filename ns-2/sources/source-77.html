36     /**
37      * Adds new column to table, which can be either empty or can have data already. When existing columns
38      * already have data this function will create ".top" file in addition to column files. ".top" file contains
39      * size of partition at the moment of column creation. It must be used to accurately position inside new
40      * column when either appending or reading.
41      *
42      * <b>Failures</b>
43      * Adding new column can fail in many situations. None of the failures affect integrity of data that is already in
44      * the table but can leave instance of TableWriter in inconsistent state. When this happens function will throw CairoError.
45      * Calling code must close TableWriter instance and open another when problems are rectified. Those problems would be
46      * either with disk or memory or both.
47      * <p>
48      * Whenever function throws CairoException application code can continue using TableWriter instance and may attempt to
49      * add columns again.
50      *
51      * <b>Transactions</b>
52      * <p>
53      * Pending transaction will be committed before function attempts to add column. Even when function is unsuccessful it may
54      * still have committed transaction.
55      *
56      * @param columnName              of column either ASCII or UTF8 encoded.
57      * @param symbolCapacity          when column columnType is SYMBOL this parameter specifies approximate capacity for symbol map.
58      *                                It should be equal to number of unique symbol values stored in the table and getting this
59      *                                value badly wrong will cause performance degradation. Must be power of 2
60      * @param symbolCacheFlag         when set to true, symbol values will be cached on Java heap.
61      * @param columnType              {@link ColumnType}
62      * @param isIndexed               configures column to be indexed or not
63      * @param indexValueBlockCapacity approximation of number of rows for single index key, must be power of 2
64      */
