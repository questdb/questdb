Per D-01 and D-02: Replace the broken Lomuto-partition quickSort/partition/swap methods in all 8 inner classes (2 inner classes per factory x 4 factories) with a correct dual-pivot quicksort adapted from `GroupByDoubleList.sort()`.

**The problem:** The current Lomuto partition uses strict `<` comparison. On all-equal elements, the pivot always lands at `right`, degrading to O(N^2). The correct fix is a three-way partition (Dutch National Flag) that skips equal elements.

**Approach for Phase 1:** Replace the broken sort inline in each inner class. Phase 4 (QUAL-01) will extract these into a shared utility. For now, correctness is the goal.

Each window factory has TWO inner classes with duplicate sort code:
1. `*OverPartitionFunction` -- sort operates on `listMemory.getDouble(listPtr + index * 8)` where `listPtr` is a per-partition offset
2. `*OverWholeResultSetFunction` -- sort operates on `listMemory.getDouble(index * 8)` (no listPtr offset, uses `this.listMemory` directly)

**For each inner class, replace the three methods (`quickSort`, `partition`, `swap`) with a correct implementation.**

The replacement sort must:
1. Handle all-equal elements without O(N^2) degradation (three-way partitioning)
2. Use median-of-three pivot selection (already partially done)
3. Fall back to insertion sort for small subarrays (threshold ~47, matching GroupByDoubleList)
4. Work with `MemoryARW` via `getDouble`/`putDouble` (not raw Unsafe pointers)

Here is the replacement implementation for the `*OverPartitionFunction` variant (with `listPtr` parameter). Adapt the method signatures to match the existing code style in each class:

```java
private static final int INSERTION_SORT_THRESHOLD = 47;

private void quickSort(long listPtr, long left, long right) {
    int length = (int) (right - left + 1);
    // Insertion sort for small arrays
    if (length < INSERTION_SORT_THRESHOLD) {
        for (long i = left + 1; i <= right; i++) {
            double key = listMemory.getDouble(listPtr + i * 8);
            long j = i - 1;
            while (j >= left && listMemory.getDouble(listPtr + j * 8) > key) {
                listMemory.putDouble(listPtr + (j + 1) * 8, listMemory.getDouble(listPtr + j * 8));
                j--;
            }
            listMemory.putDouble(listPtr + (j + 1) * 8, key);
        }
        return;
    }

    // Three-way partition (Dutch National Flag) for handling equal elements
    // Choose pivot as median of three
    long mid = left + (right - left) / 2;
    if (listMemory.getDouble(listPtr + left * 8) > listMemory.getDouble(listPtr + mid * 8)) {
        swap(listPtr, left, mid);
    }
    if (listMemory.getDouble(listPtr + left * 8) > listMemory.getDouble(listPtr + right * 8)) {
        swap(listPtr, left, right);
    }
    if (listMemory.getDouble(listPtr + mid * 8) > listMemory.getDouble(listPtr + right * 8)) {
        swap(listPtr, mid, right);
    }
    double pivot = listMemory.getDouble(listPtr + mid * 8);

    // DNF partition: elements < pivot | elements == pivot | elements > pivot
    long lt = left;   // listMemory[left..lt-1] < pivot
    long gt = right;  // listMemory[gt+1..right] > pivot
    long i = left;    // listMemory[lt..i-1] == pivot

    while (i <= gt) {
        double val = listMemory.getDouble(listPtr + i * 8);
        if (val < pivot) {
            swap(listPtr, lt, i);
            lt++;
            i++;
        } else if (val > pivot) {
            swap(listPtr, i, gt);
            gt--;
        } else {
            i++;
        }
    }

    // Recurse on < pivot and > pivot partitions (skip == pivot partition)
    if (lt > left) {
        quickSort(listPtr, left, lt - 1);
    }
    if (gt < right) {
        quickSort(listPtr, gt + 1, right);
    }
}

private void swap(long listPtr, long i, long j) {
    double temp = listMemory.getDouble(listPtr + i * 8);
    listMemory.putDouble(listPtr + i * 8, listMemory.getDouble(listPtr + j * 8));
    listMemory.putDouble(listPtr + j * 8, temp);
}
```
